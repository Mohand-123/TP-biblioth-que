# Projet Bibliothèque - TP1

## Structure du projet

```
tp1C/
├── inc/              # Fichiers d'en-tête (.h)
├── src/              # Fichiers sources (.c)
├── obj/              # Fichiers objets (.o)
├── lib/              # Bibliothèques
├── exe/              # Exécutables
├── test/             # Tests
├── test_unitaire/    # Tests unitaires
├── doc/              # Documentation
├── outils/           # Outils et scripts
├── Makefile          # Compilation
└── README.md         # Ce fichier
```

## Compilation

```bash
make        # Compiler le projet
make run    # Compiler et exécuter
make clean  # Nettoyer les fichiers de build
```

## Description

Système de gestion de bibliothèque en langage C.

### Fonctionnalités
- Afficher les informations de la bibliothèque
- Ajouter un livre
- Rechercher un livre
